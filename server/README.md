# Running our server 
1. Open up a terminal and cd to:
```bash
..\portia\server
```
2. Then run:
```bash
node server.js
```
You should recieve a message to the terminal saying the connection succeeded

3. Open up another terminal and cd to:
```bash
..\portia
```
4. Then run 
```bash 
npm start 
```
This should open up a window with our react app running
